Autiva – Fahrzeug-Check App

Was ist das?

Eine kleine App für Android und Windows, mit der man Fahrzeug-Prüfungen digital erfassen kann. Anstatt Zettel auszufüllen, gibt man die Daten (Reifen, Öl, Mängel) direkt in die App ein und kann am Ende einen Bericht erstellen.

Funktionen
Fahrzeuge verwalten: Autos anlegen und in einer Liste speichern.

Prüfbericht erstellen: Schritt-für-Schritt-Abfrage von Reifendruck, Profiltiefe, Ölstand und Warnleuchten.

PDF-Export: Die Ergebnisse werden direkt als PDF-Datei gespeichert.

Datenbank: Alle Infos bleiben lokal auf dem Gerät gespeichert (SQLite).

Technik-Stack
Framework: .NET 10 / MAUI

Sprache: C# / XAML

Datenbank: SQLite

PDF-Erzeugung: Syncfusion (Net.Maui)
